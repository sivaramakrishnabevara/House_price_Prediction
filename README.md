# House Price Prediction

A Machine Learning project that predicts house prices based on property features such as location, area, bedrooms, bathrooms, latitude, longitude, and other housing attributes.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Feature Engineering
4. Exploratory Data Analysis (EDA)
5. Model Training
6. Model Evaluation
7. House Price Prediction

## Results
Built and evaluated regression models to accurately predict house prices using housing market data.
